Interface IEvent

Defines the basics of an event.

Namespace:Bifrost.Events
Assembly:Bifrost.dll
Syntax
public interface IEvent : IConvention
Remarks

Types inheriting from this interface can be used in event sourcing and will be picked up by the event migration system. You most likely want to subclass Event.
