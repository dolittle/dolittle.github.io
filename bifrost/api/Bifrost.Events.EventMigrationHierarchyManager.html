public class EventMigrationHierarchyManager : IEventMigrationHierarchyManager
Remarks

The manager will automatically build an EventMigrationHierarchy for all events and allow clients to query for the current migration level for a specific logical event or the concrete type of a particular link in the migration chain for a logical event.
