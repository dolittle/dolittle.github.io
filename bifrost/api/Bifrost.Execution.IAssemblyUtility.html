Remarks

The need for this is questionable - the interface System.Runtime.InteropServices._Assembly does not have the IsDynamic property as the implementation System.Reflection.Assembly has. This might go away as there has been a realization that System.Runtime.InteropServices._Assembly might not be needed, it was introduced to do testing easier. Turns out however that the implementation System.Reflection.Assembly has pretty much everything virtual.
